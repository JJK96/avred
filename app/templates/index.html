avred

antivirus reducer - AV signature identificator

about this

This will split the provided file until Windows Defender does not detect it anymore. It will show which parts of the file is being detected.

Supported formats:

  • .exe (PE or .NET)
  • .docm
  • .ps1

Analyze your own file.

All uploads will be stored. All requests are logged. Defender sample submission is disabled. Use at your own risk. Do not misuse this service.
